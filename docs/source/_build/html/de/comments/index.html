12. Comments¶

For the categories „Projects“, „Contracts“ and „Employee“ it´s possible to save comments. Comments are build out of a free text and the possibility, to add some files. Comments are generally just assigned to a object and save the author and the publishing date automatically.

Comments own a simple formula, which contains a field for free text and a dialogue to umpload some files. Comments always appear in the detail view from the object, where they was crated for.
